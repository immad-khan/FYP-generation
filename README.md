# AI FYP Idea Generator

An intelligent web application that generates personalized Final Year Project ideas for university students using AI.

## Features

### For Students
- 🔐 Secure authentication and profile management
- 🎓 Academic profile input (department, CGPA, semester)
- 🤖 AI-powered idea generation based on interests
- 💾 Save and manage favorite ideas
- 📊 Semester record management
- 🔍 Search and filter saved ideas

### For Faculty
- 👨‍🏫 Dashboard with student overview
- 👥 Student profile management
- 💡 Generate ideas for specific students
- 📋 View student saved ideas
- 👤 Faculty profile management

## Tech Stack

### Frontend
- React.js 18
- React Router v6
- Tailwind CSS
- Axios for API calls
- Context API for state management

### Backend
- Node.js with Express
- MySQL database
- JWT authentication
- WebSocket for real-time updates

## Prerequisites

- Node.js 16+ and npm
- MySQL 8.0+
- Git
